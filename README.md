# The Tightening

Interactive chord-progression graph with real-time MIDI detection.

Build progressions, visualize chord relationships on a force-directed graph seeded by the circle of fifths, and play chords on a MIDI keyboard to see where you are and where you can go next.

## Download

**[Latest release](https://github.com/kenEldridge/the-tightening/releases/latest)** — grab the `.exe` installer for Windows.

## Dev setup

```
npm install
npm run dev
```

## Build installer locally

```
npm run dist
```

Outputs to `release/`.
